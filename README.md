〈プログラムの目的〉

書籍を登録・一覧表示・削除できる簡易的な図書管理アプリです。  
オブジェクト指向（クラス分割 / カプセル化）の基本を学ぶ目的で作成しました。

〈実行方法〉

1.src ディレクトリへ移動
cd src

2.コンパイルjavac Book.java Library.java LibraryApp.java

3.実行
java LibraryApp

〈クラスの役割と設計のポイント〉

-Book クラス-
・1 冊の本の「タイトル/著者/出版年」をフィールドとして保持
・アクセス修飾子と getter でカプセル化したデータ取得
・書籍情報を文字列として返す toString メソッド

-Library クラス-

・ArrayList を使って書籍を管理
・書籍の登録、一覧表示、削除、(発展要素)検索機能の追加

-LibraryApp クラス-

・メニューの表示
・ユーザ入力の受け取り
・Library への操作命令
・書籍登録時の Book の生成
・無限ループを使って継続的な操作を可能にする

-設計ポイント-

Book は書籍の雛型としてデータだけを持ち、Library でその書籍を管理するLibraryApp はそれらを使ったユーザーとのやり取り全般を担当。というようにデータ、管理、実行を明確に分担して設計することでコードが読みやく、分かり易い構造になるよう、工夫しました。

〈追加機能の実装〉

-入力チェック-

タイトル・著者が空文字で登録されたときに再入力を促す。
出版年に入力されたものが数字以外でも処理が中断されないようにする。
(NumberFormatException)を利用した例外処理で安全に再入力できるようにしました。

-検索機能-

キーワードを入力すると、該当する書籍を一覧表示する。
部分一致 String.contains() を使って検索の柔軟性を高めました。
